# Quick Start Scripts

## Development

### Run everything (frontend + backend)
npm run dev

### Frontend only
npm run dev:frontend

### Backend only
npm run dev:backend

## Production

### Build
npm run build

### Start production server
npm start

## Installation

### First time setup
npm run install:all

## Deployment

### GitHub Pages
git push origin main
(Auto-deploys via .github/workflows/deploy.yml)
